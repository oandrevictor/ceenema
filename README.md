# README

This is a Movielist Web App.
Use of IMDB API to fetch movie data.
Avaliable @ http://ceenema.herokuapp.com/

Dependencies:

* Ruby  (2.3.3)
* Rails (5.1.4)
* Node (6.11)
* Postgresql <br />
** Use npm to:
* Bower <br />
*Use bower to:
** Angular
To install dependencies: 
  rake run bower:install

To migrate the database:
  rake db:migrate

To run:
 rails s
