# README

This is an Movielist Web App.
Avaliable @ http://ceenema.herokuapp.com/

Dependencies:

* Ruby  (2.3.3)
* Rails (5.1.4)
* Node (6.11)
* Postgresql
Use npm to:
* Bower
Use bower to:
* Angular
To install dependencies: 
  rake run bower:install

To migrate the database:
  rake db:migrate

To run:
 rails s
